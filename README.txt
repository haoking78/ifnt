使用方式（GitHub Pages）
1) 進入你的 ifnt repo（例如 haoking78/ifnt）。
2) 將此 zip 內的所有檔案覆蓋至 repo 根目錄（或手動上傳 6 個檔案）。
   - index.html
   - app.js
   - service-worker.js
   - manifest.json
   - app_icon_192.png
   - app_icon_512.png
3) Commit 到 main 分支，等待 GitHub Pages 完成部署。
4) 用以下網址開啟（帶版本參數可避開快取）：
   https://haoking78.github.io/ifnt/?v=v6.1.6-20251031-2

注意：所有連結均為相對路徑（./），可確保在 /ifnt/ 子路徑正常載入。
